
# 智能家居中控系统 - 前端

这是一个基于 React 的智能家居控制面板，配合 Python 后端使用。

## 功能

- **设备状态监控**：实时显示灯光、空调、窗户和温度状态。
- **智能对话助手**：支持自然语言控制（如"帮我把空调打开"）。
- **手动控制**：点击设备卡片可直接切换开关状态。

## 启动方式

1. 确保后端服务已启动（运行在 8000 端口）。
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动开发服务器：
   ```bash
   npm run dev
   ```
4. 浏览器访问：http://localhost:5173

## 技术栈

- React
- Vite
- Axios
- React Icons
